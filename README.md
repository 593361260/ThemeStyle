# ThemeStyle
demo的主要作用是仿网易云音乐的颜色值切换皮肤, 适配6.0以下状态栏文字内容颜色不能更改, 实现原理是占位的方式, 计算出状态栏的高度,然后更改状态栏的颜色,

##换肤框架 
``https://github.com/ximsfei/Android-skin-support``
